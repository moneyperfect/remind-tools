# 🎯 任务管理 API 测试项目

> 使用 **MiniMax-M2.5** 模型生成

## 📋 测试需求

### 原有 5 点
1. ✅ 用户注册/登录（JWT 认证）
2. ✅ 创建任务（标题、描述、截止时间、优先级）
3. ✅ 查看任务列表（支持筛选、排序、分页）
4. ✅ 更新/删除任务
5. ✅ 任务提醒（定时任务模拟）

### 新增第 6 点
6. ✅ **页面美观性** - 现代深色主题、动画效果、响应式设计

---

## 🚀 快速启动

### 1. 安装依赖
```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动服务
```bash
cd backend
python main.py
```

### 3. 访问
- 前端：http://localhost:8000
- API 文档：http://localhost:8000/docs

## ✨ 特性

- **JWT 认证** - 安全登录
- **完整 CRUD** - 创建、读取、更新、删除
- **筛选排序** - 按状态、优先级、创建时间筛选和排序
- **分页** - 10条/页
- **任务统计** - 总数、待完成、已完成、紧急
- **精美 UI** - 深色主题、动画、平滑过渡
- **响应式** - 支持手机/平板
- **即时反馈** - Toast 通知
